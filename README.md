# Assets folder

Put images, logos, and other media files here.

## Files included

- `favicon.svg` — the small icon shown in the browser tab. Replace with your logo (SVG or PNG, square).
- `hero-placeholder.svg` — placeholder image. Not currently used by the site, but you can drop it into `index.html` if you want a background image.

## How to use your own image

1. Drop your image file into this folder (e.g. `my-photo.jpg`).
2. In `index.html`, reference it like:
   ```html
   <img src="assets/my-photo.jpg" alt="Description of the image" />
   ```

## Tips

- Use JPG for photos, PNG for graphics with transparency, SVG for logos/icons.
- Keep image files under ~500 KB so the site loads fast. Use [tinypng.com](https://tinypng.com) to compress.
